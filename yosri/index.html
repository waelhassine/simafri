QUESTIONS FRÉQUENTES

Que se passe-t-il après les 2 années d'hébergement web offert ?

Après 2 ans, plusieurs options s’offrent à vous :

  • Si vous êtes toujours étudiant et que vous avez toujours moins de 25 ans, vous pourrez demander le renouvellement de votre offre gratuite pour 2 années supplémentaires.
  • Si vous n’êtes plus étudiant ou que vous avez plus de 25 ans mais que vous souhaitez tout de même continuer à utiliser votre offre, vous pourrez régler l’abonnement à un tarif très préférentiel (tarif indicatif : 20 000 Ar / an).
  • Vous pourrez faire évoluer votre offre vers un pack supérieur (Pack Perso, Pack Pro ou Pack Premium) au tarif public.
  • Si vous ne choisissez aucune des options précédentes, votre hébergement web sera automatiquement clôturé sans frais.

Y a-t-il de la publicité que je devrai mettre sur mon site pour Simafri ?

Il n’y a aucune publicité à mettre sur votre site web. Simafri agit de manière totalement désintéressée, dans le but d’aider le développement du pays. création de leur site Internet a la même priorité. Habituellement Il faut compter entre 1 à 3 semaines suivant la disponibilité de nos équipes et la complexité du site web à réaliser. Dans tous les cas, vous serez tenus régulièrement informés de l’évolution de votre projet et vous pourrez solliciter notre service client à tout moment.

Quelles sont les différences entre le pack Web Mpianatra et les packs d'hébergement web classiques ?

Les seules différences se situent dans la limitation de certaines caractéristiques comme l’espace de stockage ou encore le nombre de comptes email. Pour le reste tout est identique. Les comptes « Web Mpianatra » sont hébergés sur les mêmes serveurs haut de gamme que nos offres payantes.
